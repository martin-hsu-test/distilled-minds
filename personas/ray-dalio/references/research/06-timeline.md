# Ray Dalio · 人生時間線調研

## 早期人生

| 時間 | 事件 | 對思維的影響 |
|------|------|------------|
| 1949年8月8日 | 出生於紐約市皇后區 Jackson Heights，父親是意大利裔爵士音樂家 | 中產工薪家庭背景，非精英出身 |
| 1957年 | 全家搬至長島 Manhasset | 進入中產郊區環境 |
| 1961年（12歲）| 買入人生第一隻股票：Northeast Airlines，當時股價 $5。買入後公司被收購，三倍獲利。 | 「早早發現市場規律比偶然性更重要」 |
| 1960年代 | 就讀 Herricks High School，成績平平，但熱愛投資和體育 | 「我在學校表現一般，但在我真正感興趣的事情上全力以赴」 |
| 1971年 | 取得長島大學（C.W. Post College）金融學士學位 | 在紐約股票交易所當文員，累積實際市場經驗 |
| 1971年 | 尼克森宣布美元脫離金本位，道瓊斯指數意外大漲（與直覺相反），第一次意識到「現實比你想象的更複雜」 | 奠定「永遠壓力測試直覺」的習慣 |
| 1972年 | 哈佛商學院就讀，1973年取得MBA | 學習系統性分析框架，結識各行業人才 |
| 1974年 | 在 Shearson Hayden Stone 任大宗商品期貨交易員，因在新年派對上醉後打了老闆一拳而被解雇 | 「衝動的代價」——此後極度重視情緒管理 |

## Bridgewater 創立期（1975-1990）

| 時間 | 事件 | 對思維的影響 |
|------|------|------------|
| 1975年 | 在曼哈頓公寓創立 Bridgewater Associates，最初是諮詢公司，為企業客戶管理匯率/利率風險 | 從零開始，建立「靠研究和分析建立信任」的文化 |
| 1981年 | 公司搬至康涅狄格州 Westport | 刻意遠離華爾街，保持獨立思考 |
| 1982年 | 押注墨西哥債務危機，公開作證預言美國銀行業危機；美聯儲干預後市場反向，Bridgewater 幾乎破產 | **職業生涯最重要的失敗**：創建原則系統的直接觸發點 |
| 1982-1986年 | 重建 Bridgewater，以教訓重建原則，麥當勞成為關鍵客戶 | 建立「從失敗中提煉原則」的習慣，公司文化成型 |
| 1987年 | 在1987年股市崩盤中獲利，首次建立外部聲譽 | 「對沖與宏觀分析的組合在危機中最有價值」 |

## 機構化期（1991-2006）

| 時間 | 事件 | 對思維的影響 |
|------|------|------------|
| 1991年 | 推出 Pure Alpha 基金，正式進入機構對沖基金管理 | 「Alpha 是可以系統化創造的，不是運氣」 |
| 1993年 | 受到高管集體備忘錄批評其管理方式（不尊重人、操控性強等）| 「痛苦+反思=進步」的親身示範，開始建立文化反饋機制 |
| 1995年 | 為美國財政部提供通膨掛鉤債券的設計建議 | 建立政府層面的信任關係 |
| 1996年 | 推出 All Weather 基金，發明風險平價策略 | 「對沖所有場景比預測場景更可靠」 |
| 2000-2003年 | 科技泡沫崩盤時 Pure Alpha 逆市獲利，AUM 快速成長 | 確立「週期性思維」的市場價值 |
| 2005年 | Bridgewater 成為全球最大對沖基金 | 規模帶來更大話語權，但也帶來更多公眾審視 |
| 2006年 | 停止接受新投資者 | 「規模是alpha的敵人」 |

## 思想輸出期（2018-至今）

| 時間 | 事件 | 對思維的影響 |
|------|------|------------|
| 2018年 | 出版《Big Debt Crises》，免費提供PDF | 建立宏觀分析學術遺產 |
| 2019年 | 「資本主義需要改革」大型LinkedIn文章 | 超越投資，進入政策影響力領域 |
| 2020年 | 長子 Devon 因車禍去世（42歲）| 「生命的無常讓我更珍視每一次對話和選擇」 |
| 2021年 | 出版《Changing World Order》 | 帝國週期框架成為最廣傳播的宏觀分析工具之一 |
| 2022年 | 通膨失準，基金表現落後；Nir Bar Dea 接任 CEO | 公開承認失誤，但堅持框架的長期有效性 |
| 2023年 | Rob Copeland《The Fund》出版；NYT 調查質疑 Bridgewater 投資系統 | 最大的公關危機，Dalio 強力反駁 |
| 2025年 | 賣出 Bridgewater 最後持股；出版《How Countries Go Broke》，NYT 暢銷書 | 正式脫離 Bridgewater，以全職思想家身份運作 |
