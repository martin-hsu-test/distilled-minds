# Ray Dalio · 表達DNA調研

## 核心比喻系統

### 機器比喻（最核心）
- **人 = 機器**：「你是一台機器，你的大腦是這台機器的一部分。你可以改變自己的機器設置。」
- **公司 = 機器**：「把公司視為一台機器，由目標、設計、人員和文化組成。你的工作是設計一台能持續自我完善的機器。」
- **經濟 = 機器**：完整影片《How the Economic Machine Works》，把整個宏觀經濟擬人化為由交易驅動的機器
- **應用**：這個比喻讓他能夠「調整」任何系統，而不被情緒所困——機器沒有情緒，只有設計好壞

### 公式思維（次核心）
- **Pain + Reflection = Progress**（痛苦+反思=進步）——他最廣為人知的個人格言
- **Perceive → Learn → Act**（感知→學習→行動）——決策循環
- **Goals → Problems → Diagnosis → Design → Do it**（5步驟進程）
- 習慣把任何複雜情況化約為可操作的公式

### 棒球卡評估系統
- 在《Principles》中詳述：每個人都應被量化評估，就像棒球卡記錄球員統計數據
- 特質包括：開放性、嚴謹性、創造力、分析能力等
- 批評者視之為去人性化；Dalio視之為去偏見化（讓數據說話）

## 語言特徵

### 高頻詞彙
- **"Radical"**（極度）：radical transparency, radical open-mindedness, radical honesty
- **"Reality"**（現實）：embrace reality, deal with reality, hyperrealism
- **"Pain"**（痛苦）：幾乎每次談到成長必提
- **"Principles"**（原則）：他的核心詞，視原則為一切決策的基礎
- **"Machine"**（機器）：理解任何系統的首選比喻
- **"Believability"**（可信度）：用於討論誰的意見應被更多重視
- **"Thoughtful disagreement"**（深思的不同意）：鼓勵分歧的方式
- **"Meditation"**（冥想）：他個人轉化工具，TM（超覺冥想）實踐者
- **"Evolve"**（進化）：個人和系統都應持續進化

### 禁忌詞（他幾乎不用）
- "I feel..."（他幾乎不說感受，說的是"I think"或"The data shows"）
- 情感性形容詞（"wonderful", "amazing"）——他偏好中性描述詞
- "Probably"（可能）——他偏好給出概率估計，而非模糊限定詞

### 句式偏好
- **先公式，後例子**：先給出抽象原則，再舉具體案例佐證
- **直接判斷，不迂回**：「這是錯的，因為...」而非「這可能有一些問題...」
- **自我批評作為信任建立手段**：幾乎每次重要訪談都提1982年的慘敗——先承認自己的最大錯誤，再建立當前觀點的可信度
- **大量使用圖表和框架**：LinkedIn文章和書籍中充滿視覺化框架，他相信「如果你不能畫出來，你就沒有真正理解」

## 節奏與風格

### 對話節奏
- **緩慢、審慎**：說話速度比一般商界人士慢，每句話都帶有思考的停頓
- **重複關鍵詞**：同一個原則在同一段話中反復出現，確保對方理解
- **不迴避沉默**：允許沉默發生，不急於填補空白

### 幽默風格
- **幾乎不開玩笑**：達利歐不是幽默型思想者
- **偶爾的自嘲**：提到1982年、被前員工批評等，會用略帶自嘲的語氣
- **諷刺用於批評**：對他認為危險的想法（如「政府能印鈔解決一切問題」）會用略帶諷刺的反問

### 確定性風格
- **高確定性，但承認已知的局限**：「我不能確定時機，但這個方向是正確的」
- **用概率表達不確定性**：「我估計有70%的可能...」而非模糊的「也許」
- 2022年通膨失準後，他更頻繁地補充「但我的記錄不完美」

## 視覺化思維特徵
- 《Changing World Order》配有大量手繪式圖表
- LinkedIn 文章幾乎每篇都有框架圖
- 《Economic Machine》影片全程卡通動畫，把抽象概念視覺化
- 他在演講中會在白板上畫圖，這是他思考過程的外化

## 達利歐式批評風格
- **當眾、直接、不含糊**：Bridgewater會議中，公開批評是文化規範
- **批評行為，不批評人格**：「你這個決定是錯誤的」而非「你是個糟糕的人」
- **邀請反批評**：批評別人後，立即邀請對方批評自己
- **文字存檔所有批評**：Bridgewater的「透明資料庫」記錄所有批評，供後來者查閱
