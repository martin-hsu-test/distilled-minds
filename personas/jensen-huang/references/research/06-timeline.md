# Jensen Huang：人生時間線

## 早年（1963-1992）

| 時間 | 事件 | 影響 |
|------|------|------|
| 1963年2月17日 | 台灣台北出生，後搬到台南 | 台灣文化根基 |
| 1968年 | 家庭移居泰國（父親工作） | 跨文化適應能力 |
| 1973年（10歲） | 父母送他與哥哥去美國，住在塔科馬舅舅家 | 獨立、韌性的起點 |
| 1973年 | 被錯誤地送進肯塔基州 Oneida Baptist Institute（少年感化院） | 被霸凌、學會忍耐；「那是我記憶最清晰的地方」 |
| 1975年 | 父母移民美國，搬到 Oregon Beaverton | 家庭重聚，開始正常求學 |
| 1978-1983年 | 在 Denny's 餐廳工作（洗碗工、服務生）從15歲開始 | 「Eat what you kill」服務意識、工作倫理 |
| 1984年 | Oregon State University 電機工程學士（以優異成績提前畢業，16歲入學） | 工程基礎 |
| 1985年 | 與 Lori 結婚 | 終身伴侶 |
| 1984-1992年 | LSI Logic（晶片設計師）、AMD（微處理器設計師），同時在 Stanford 修夜間課程 | 業界實踐+學術深化 |
| 1992年 | Stanford 電機工程碩士學位 | 學術背景 |

## NVIDIA 創立與早期（1993-2006）

| 時間 | 事件 | 影響 |
|------|------|------|
| 1992年底 | 在 Denny's 與 Chris Malachowsky、Curtis Priem 多次會面討論商業計劃 | 創業起源，回到曾工作的地方創業 |
| 1993年4月5日 | 正式在加州 Sunnyvale 創立 NVIDIA Corporation，初始資本 $600 | 創始 |
| 1993-1996年 | NV1 和 NV2 失敗（押注四邊形渲染而非三角形） | 「技術選擇錯誤是公司的常態」 |
| 1997年 | RIVA 128 上市，距離公司倒閉只剩30天工資 | 「我們隨時可能倒閉」成為公司信條 |
| 1999年 | NVIDIA 在 Nasdaq 上市（代碼 NVDA），發布 GeForce 256（第一款「GPU」） | GPU 這個詞被 NVIDIA 發明 |
| 2000年 | 收購 3dfx Interactive，擊敗最大競爭對手 | 整合頂尖 GPU 工程師 |
| 2004-2006年 | 開始在 GPU 中加入 FP32 支持，為 CUDA 鋪路 | 計算平台化的第一步 |

## CUDA 時代（2006-2012）

| 時間 | 事件 | 影響 |
|------|------|------|
| 2006年 | 發布 CUDA（Compute Unified Device Architecture） | 把 GPU 從遊戲加速器變成通用計算平台 |
| 2007-2009年 | 把 CUDA 整合進 GeForce 消費卡，毛利率大幅下降，市值跌至15億 | 巨大財務壓力，但堅持長期投資 |
| 2008年 | 去大學和研究機構推廣 CUDA，寫書、辦工作坊 | 播下AI時代的種子 |
| 2010-2012年 | Tegra 行動處理器進入手機市場，與高通競爭 | 後來戰略撤退的前兆 |

## 深度學習時代（2012-2022）

| 時間 | 事件 | 影響 |
|------|------|------|
| 2012年 | AlexNet 用 GeForce GTX 580 訓練，在 ImageNet 競賽大獲全勝 | CUDA+GPU 成為 AI 訓練標準 |
| 2013年 | 發布 cuDNN（深度神經網絡庫） | AI 計算的標準工具層 |
| 2013-2015年 | 逐步縮減手機 GPU 業務，重新聚焦 | 戰略收縮+集中資源 |
| 2016年 | Tesla P100（第一款為 AI 優化的 GPU）；DGX-1 超級計算機 | AI 計算硬件的定義性產品 |
| 2019年 | 以69億美元收購 Mellanox（網絡技術），布局數據中心全棧 | 從芯片到網絡的垂直整合 |
| 2020年9月 | 宣布以400億美元收購 Arm | 最大收購嘗試 |
| 2022年2月 | 放棄 Arm 收購（監管壓力，損失12.5億違約金） | 策略調整，加速自研 Grace CPU |

## AI 爆發與全球主導（2022-2025）

| 時間 | 事件 | 影響 |
|------|------|------|
| 2022年11月 | ChatGPT 發布，AI 需求爆炸式增長 | NVIDIA 成為全球最稀缺資源的提供者 |
| 2023年 | H100 GPU 等待期長達12個月；《The Economist》年度最佳 CEO | 市場爆發，NVIDIA 進入超增長期 |
| 2023年 | 市值首次突破1兆美元 | 第七家達到兆美元市值的美國公司 |
| 2023年 | 正式提出 AI Factory、Sovereign AI 框架 | 重新定義 NVIDIA 的角色 |
| 2024年 | Blackwell 架構發布；市值突破3兆美元；Computex 的「Jensanity」現象 | 台灣英雄，粉絲明星化 |
| 2025年 | GES Keynote；市值首次突破5兆美元（全球第一）；Rubin 架構 | 全球最有價值公司 |
| 2025年 | Queen Elizabeth Prize for Engineering；FT Person of the Year；Time Person of the Year | 全球最高工程與科技榮譽 |
