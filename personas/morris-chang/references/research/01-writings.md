# 01 - 張忠謀：著作與系統性思考

> 調研維度：書籍、長文、演講稿中反覆出現的核心論點

---

## 一、主要著作

### 1.《張忠謀自傳（上冊）》（1998年，天下文化出版）
- **覆蓋範圍**：1931年出生到1964年，聚焦早年成長與進入半導體產業
- **核心主題**：
  - 戰亂童年如何塑造韌性——從寧波到香港到重慶，6歲前搬家超過6次
  - 哈佛人文教育的深遠影響——「哈佛一年，是我一輩子知識的骨幹」
  - MIT工程訓練的嚴謹思維養成
  - 意外進入半導體產業（原本想讀博士，因經濟壓力先工作）
  - 在Sylvania和TI的早期技術管理經驗
- **可信度**：一手來源（本人撰寫）

### 2.《張忠謀自傳（下冊）》（2024年12月出版，天下文化）
- **覆蓋範圍**：1964年之後至退休，涵蓋TI高層生涯、創辦台積電全過程
- **核心主題**：
  - 在TI的權力鬥爭與離開——「我在TI的最後幾年，已經感覺到天花板」
  - 創辦台積電的完整決策過程
  - 晶圓代工模式（pure-play foundry）的商業邏輯推演
  - 與飛利浦合資的談判細節
  - 對半導體產業未來的長期判斷
  - 退休、回鍋、再退休的心路歷程
- **可信度**：一手來源（本人撰寫），但注意自傳必然有自我敘事篩選

## 二、反覆出現的核心論點（≥3次=真信念）

### 論點1：「晶圓代工是半導體產業的必然演化」
- **出處**：自傳、多次演講、股東信
- **論證邏輯**：
  1. IC設計和IC製造是兩種完全不同的能力
  2. 製造的資本密度越來越高，不是每家公司都負擔得起
  3. 專業分工讓設計公司（fabless）專注創新，代工廠專注製程
  4. 這不是權宜之計，是產業結構的進化方向
- **原話**：「我看到一個產業結構性的機會——把製造從設計中分離出來，讓兩邊都能做到最好。」（多次演講中的大意）
- **頻率**：幾乎每次重要演講都會闡述
- **可信度**：一手

### 論點2：「技術領先是唯一可持續的競爭優勢」
- **出處**：股東信、產業論壇發言、訪談
- **論證邏輯**：
  1. 成熟製程的利潤空間會被壓縮
  2. 客戶（Apple、Qualcomm、NVIDIA）需要的是最先進製程
  3. 技術領先帶來定價權——「我們不打價格戰」
  4. 研發投入是護城河，不是成本
- **張忠謀的表述**：「我們的策略很清楚，就是要在技術上領先。不是領先一點點，是要領先夠多，讓競爭對手追不上。」
- **頻率**：貫穿整個經營生涯
- **可信度**：一手（多次公開發言）

### 論點3：「定價權來自不可替代性」
- **出處**：投資人會議、訪談
- **論證邏輯**：
  1. 台積電不追求市佔率最大化，追求利潤最大化
  2. 「如果客戶有其他一樣好的選擇，我們的定價就站不住腳」
  3. 先進製程的不可替代性 = 定價權
  4. 這與他在TI學到的經驗相反——TI曾打過價格戰，結果不好
- **頻率**：在討論商業模式時反覆出現
- **可信度**：一手

### 論點4：「誠信和信任是代工模式的根基」
- **出處**：自傳、演講、訪談
- **論證邏輯**：
  1. 代工廠必須值得信任——客戶把最機密的設計交給你
  2. 台積電絕不設計自己的IC，以避免與客戶競爭（與Samsung形成對比）
  3. 「客戶信任我們不會偷他們的設計」——這是商業承諾，也是企業文化
  4. 信任的建立需要時間，破壞只需一次
- **頻率**：每次解釋台積電vs三星的差異時都會提到
- **可信度**：一手

### 論點5：「半導體是國家戰略資產」
- **出處**：近年（2020後）的公開發言、APEC場合
- **論證邏輯**：
  1. 半導體已經不只是商品，是國家安全的關鍵要素
  2. 台灣在先進製程的主導地位是「矽盾」（Silicon Shield）
  3. 全球化正在倒退，半導體供應鏈會被政治化
  4. 「去全球化對所有人都不好，但它正在發生」
- **值得注意**：這個論點有時間性演變。2018年前張忠謀更傾向全球化觀點；2020年後明顯轉向認為地緣政治會重塑產業
- **可信度**：一手（公開演講和媒體訪問）

### 論點6：「人才是最重要的資產——但必須是對的人」
- **出處**：自傳、內部講話、訪談
- **論證邏輯**：
  1. 半導體產業的核心競爭力在人
  2. 「我花最多時間的事情之一，就是找人、評估人、培養人」
  3. 選擇繼任者是CEO最重要的決定之一
  4. 但不是所有優秀的人都適合領導台積電——需要策略眼光，不只是執行力
  5. 蔡力行事件（2009）暗示他對「對的人」有非常具體的標準
- **可信度**：一手+二手混合

## 三、自創術語與概念

| 術語 | 含義 | 首次提出/使用語境 |
|------|------|-----------------|
| Pure-play foundry（專業晶圓代工） | 只做代工、不做自有品牌IC的商業模式 | 1987年創辦台積電時的定位論述 |
| 「大聯盟」vs「小聯盟」 | 用來比喻先進製程（大聯盟）vs成熟製程（小聯盟）的競爭格局 | 多次演講 |
| 「矽盾」引用與延伸 | 台灣半導體產業作為地緣政治保護 | 2020年後頻繁引用 |
| 「學習曲線」的經營應用 | 製造良率的學習曲線是核心競爭力——先量產者的良率優勢會自我強化 | TI時期即有此認識 |

## 四、智識譜系線索（從著作中推斷）

### 影響他的書籍和思想家
- **彼得·杜拉克（Peter Drucker）**：管理思想的核心影響——張忠謀多次引用杜拉克的管理理念
- **安迪·乖洛夫（Andy Grove）**：《Only the Paranoid Survive》——張忠謀與Grove有長期交情，但在經營哲學上有顯著差異
- **哈佛人文教育**：莎士比亞、經濟學、哲學——張忠謀自認哈佛一年的人文訓練比MIT的工程訓練對他影響更深
- **古典音樂**：莫扎特、貝多芬——不直接影響商業思維，但反映他對卓越和紀律的追求

### 他可能不認同的思想
- 「快速失敗」（fail fast）文化——張忠謀強調深思熟慮後再行動
- 短期主義——他的策略視野通常以5-10年為單位
- 「破壞式創新」（disruptive innovation）——台積電的成功更像是漸進式的技術積累，而非顛覆

## 五、矛盾與張力

### 矛盾1：全球化信仰 vs 地緣政治現實主義
- **早期**（2018前）：強烈支持全球化，認為自由貿易對所有人有利
- **近期**（2020後）：承認全球化正在倒退，半導體已被政治化
- **張力**：他內心可能仍信仰全球化，但理智上接受了現實

### 矛盾2：重視人才 vs 高壓管理風格
- **主張**：「人才是最重要的」「要給人發展空間」
- **實際**：台積電以高壓、高強度著稱；蔡力行事件顯示他對不符標準的繼任者會果斷更換
- **可能的調和**：在他的認知中，高標準本身就是重視人才的表現

### 矛盾3：技術理想主義 vs 商業現實主義
- **他既是工程師出身（MIT/Stanford），又是商業領導者（TI副總裁、台積電CEO）**
- 在技術方向上追求極致，但在商業決策上極度務實（不做賠錢的事）
