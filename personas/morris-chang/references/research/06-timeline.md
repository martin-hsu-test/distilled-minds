# 06 - 張忠謀：人生時間線

> 調研維度：完整人生軌跡、關鍵節點、思想轉折點

---

## 早年（1931-1949）

| 時間 | 事件 | 對思維的影響 |
|------|------|------------|
| 1931年 | 出生於浙江寧波，父親是財政局官員 | 中上階層家庭背景，重視教育 |
| 1931-1937 | 在南京度過幼年 | — |
| 1937年 | 日本侵華，全家遷移——南京→廣州→香港 | 戰亂中的生存經驗，培養適應力 |
| 1937-1941 | 在香港英文學校就讀 | 英文啟蒙，西式教育的第一次接觸 |
| 1941年 | 太平洋戰爭爆發，香港淪陷 | 再次經歷戰爭動盪 |
| 1941-1943 | 隨家人輾轉到重慶 | 長途遷移中的韌性培養 |
| 1943-1948 | 在重慶南開中學就讀 | 受到嚴格的傳統教育訓練 |
| 1948年 | 隨叔父赴香港 | — |
| 1949年 | 18歲，赴美國求學 | **重大轉折**：離開中國，開始美國生涯 |

### 思想轉折：「我的人格在戰亂中成形——適應變化、獨立思考、不依賴任何不可靠的東西。」

---

## 美國求學（1949-1955）

| 時間 | 事件 | 對思維的影響 |
|------|------|------------|
| 1949-1950 | 哈佛大學一年級（人文教育） | **關鍵影響**：「哈佛一年，是我一輩子知識的骨幹」——人文素養、批判思維 |
| 1950年 | 轉學至MIT（麻省理工學院） | 轉向工程，因為「哈佛太貴，MIT有獎學金」（也有學術興趣的因素） |
| 1952年 | MIT機械工程學士 | 嚴謹的工程思維訓練 |
| 1953年 | MIT機械工程碩士 | — |
| 1953-1955 | 試圖讀博士（MIT），未完成 | 博士資格考試未通過（第二次嘗試），這是他少有的公開「失敗」經驗 |

### 思想轉折：博士考試失敗迫使他提前進入工業界——「這是我人生最大的挫折之一，但也讓我走上了另一條路。」

---

## 早期工業生涯（1955-1958）

| 時間 | 事件 | 對思維的影響 |
|------|------|------------|
| 1955年 | 加入Sylvania公司半導體部門 | 意外進入半導體產業——原本想做機械工程 |
| 1955-1958 | 在Sylvania做半導體研發 | 第一次接觸半導體技術；學到了「製造的學習曲線」概念 |

### 思想轉折：「我原本對半導體一無所知，但很快發現這是一個充滿機會的新產業。」

---

## 德州儀器時期（1958-1983）

| 時間 | 事件 | 對思維的影響 |
|------|------|------------|
| 1958年 | 加入德州儀器（TI） | 進入全球頂尖半導體公司 |
| 1958-1964 | IC研發和生產管理 | 深入理解半導體製造——良率、學習曲線、規模經濟 |
| 1964年 | 升任TI積體電路部門總經理 | 第一次管理大型團隊的經驗 |
| 1967年 | 取得史丹佛大學電機博士 | 在職進修完成博士——「我用了14年完成當年沒完成的事」 |
| 1972年 | 升任TI集團副總裁 | 進入公司最高管理層 |
| 1978年 | 升任TI資深副總裁 | 管理全球半導體業務 |
| 1983年 | 離開TI | **關鍵轉折**：被跳過CEO接班人選，加上策略分歧 |

### 思想轉折：
- 在TI學到：大公司的政治和策略限制可以壓垮一個人的抱負
- 學到經營管理的核心能力——但也學到了「不想做的事」
- 「25年的TI經驗教會我如何經營一家半導體公司，也教會我什麼是錯的經營方式。」

## 過渡期（1983-1987）

| 時間 | 事件 | 對思維的影響 |
|------|------|------------|
| 1984年 | 任General Instrument總裁 | 短暫——「這不是我想做的事」 |
| 1985年 | 接受台灣政府邀請，擔任工研院院長 | **關鍵轉折**：決定回台灣，開始構思半導體產業的新模式 |
| 1985-1987 | 在工研院期間構思晶圓代工模式 | 所有在TI的經驗、對產業結構的觀察，在這個時期整合成一個清晰的商業構想 |
| 1987年2月 | 台積電成立 | 56歲創業——「我比大多數創業者老得多，但我也比他們看了更多」 |

### 思想轉折：「回台灣不是退路，是新的起跑線。我在美國做到了能做的一切，現在是做自己的事的時候。」

---

## 台積電創業與成長（1987-2005）

| 時間 | 事件 | 對思維的影響 |
|------|------|------------|
| 1987年 | 台積電成立，飛利浦為大股東 | 創業初期靠飛利浦訂單存活 |
| 1988-1993 | 爭取fabless公司客戶 | 逐步驗證代工模式——每一個新客戶都是證明 |
| 1994年 | 台積電在台灣證交所上市 | 公司公開化 |
| 1997年 | 台積電在紐約證交所上市（ADR） | 進入國際資本市場 |
| 1990年代末 | 超越聯電，成為全球最大晶圓代工廠 | 代工模式的最終驗證 |
| 2000年 | 網路泡沫——台積電受衝擊但存活 | 「經濟週期是產業常態，關鍵是在低谷時保持投資」 |
| 2003-2005 | 與IBM合作開發先進製程 | 技術策略的深化 |
| 2005年 | 第一次退休，交棒蔡力行 | 73歲退休 |

## 退休、回鍋與再退休（2005-2018）

| 時間 | 事件 | 對思維的影響 |
|------|------|------------|
| 2005年 | 退休，任董事長但不管日常營運 | 嘗試放手 |
| 2008年 | 全球金融海嘯，蔡力行大幅裁員 | 觸發回鍋的導火線 |
| 2009年6月 | 回任CEO | **關鍵轉折**：「我不應該把公司交給一個在危機中會裁員的人」 |
| 2010-2012 | 搶下蘋果iPhone處理器訂單 | 最重要的客戶關係——改變了台積電的戰略地位 |
| 2012-2014 | 推進28nm、20nm、16nm FinFET | 技術領先策略的全面執行 |
| 2014-2016 | 超越Intel成為製程技術領導者 | 歷史性超越——IDM模式不再是技術最前沿 |
| 2017年 | 宣布2018年退休 | 這次是有計劃的、平穩的交接 |
| 2018年6月 | 正式退休，劉德音任董事長、魏哲家任CEO | 86歲正式離開管理崗位 |

### 思想轉折：「第二次經營台積電，我更清楚什麼是重要的——技術領先、人才、客戶信任。其他都是噪音。」

## 個人生活

### 家庭
- **第一任妻子**：早年結婚，後離婚（較少公開談及）
- **第二任妻子張淑芬**：2001年結婚；張淑芬是藝術家和慈善家，活躍於台積電慈善基金會
- **子女**：有三名子女（來自第一段婚姻），但極少在公開場合談論

### 興趣愛好
- **古典音樂**：深度愛好者，特別喜愛莫扎特和貝多芬；常出席音樂會
- **橋牌**：長期橋牌愛好者——「橋牌教會我在不完全資訊下做決策」
- **閱讀**：廣泛閱讀，涵蓋歷史、傳記、經濟學、文學
- **運動**：年輕時打網球，後來步行健身
- **旅行**：退休後與張淑芬一起旅行

### 生活哲學
- 「工作和生活要有平衡」——雖然他自己在任時很難做到
- 重視健康和長壽——生活規律、飲食節制
- 「退休後最重要的是有事做——但不是被工作綁住，而是自己選擇做的事」

3. **博士考試失敗**（1955）→ 面對挫折的韌性、轉向實踐

5. **被跳過CEO**（1983）→ 自己做自己的CEO
