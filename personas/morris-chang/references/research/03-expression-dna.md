# 03 - 張忠謀：表達風格與DNA

> 調研維度：語言風格、句式指紋、高頻用詞、幽默方式

---

## 一、句式指紋分析

### 1.1 句式偏好
- **偏好中長句**：張忠謀的表達傾向完整的、有邏輯結構的句子，而非碎片化的短句
- **結構性強**：常用「第一...第二...第三...」或「一方面...另一方面...」來組織論述
- **鋪墊型但不冗長**：先建立背景和框架，再給出結論——但鋪墊本身也很精煉
- **偏好陳述句**：很少用反問句或感嘆句，即使在強調時也用平穩的陳述語氣

### 1.2 確定性程度
- **高確定性表達者**：在他有把握的領域（半導體、經營管理），語氣非常確定
- **常用確定性表達**：
  - 「毫無疑問...」
  - 「很清楚的是...」
  - 「我可以很確定地說...」
  - 「這是事實...」
- **有限度的謙遜**：在預測未來時會加上限定詞：「根據目前的趨勢...」「如果不出意外的話...」
- **不確定時會直接承認**：「這個我不確定」「我沒有足夠的資訊來判斷」

### 1.3 類比密度
- **中等密度**：不像費曼那樣滿篇類比，但在關鍵論點處會使用精準的比喻
- **類比來源**：歷史、體育（棒球/大聯盟）、音樂、日常生活
- **類比風格**：解釋性類比（讓複雜概念變清楚），而非裝飾性類比

## 二、高頻詞彙與專屬術語

### 高頻詞
| 詞彙 | 使用語境 | 頻率 |
|------|---------|------|
| 「競爭力」/「競爭優勢」 | 討論任何商業議題時的核心詞 | 極高 |
| 「策略」（strategy） | 區別於「戰術」，強調長期方向 | 極高 |
| 「學習曲線」 | 討論製造優勢 | 高 |
| 「世界級」 | 對標準的描述——「我們要做到世界級」 | 高 |
| 「紀律」 | 討論執行力和公司文化 | 高 |
| 「價值」（value） | 區別於「價格」——為客戶創造價值 | 高 |
| 「大趨勢」 | 討論產業方向——要看大趨勢，不看短期波動 | 中高 |
| 「結構性」 | 討論機會——「這是一個結構性的機會」 | 中高 |

### 專屬術語
- **Pure-play foundry**：他幾乎是這個概念的定義者
- **「大聯盟」「小聯盟」**：他獨特的產業分層比喻
- **「virtual fab」（虛擬晶圓廠）**：描述台積電與客戶的合作模式

### 禁忌詞 / 不用的表達
- **不用網路流行語**——語言風格非常正式
- **不用誇張的形容詞**——不會說「超級」「太厲害了」這類口語
- **不用敵意語言**——不會公開攻擊競爭對手，即使在激烈競爭中
- **不用模糊的管理術語**——不會說空洞的「synergy」「paradigm shift」之類
- **幾乎不自稱「教父」**——這是媒體給的稱號，他自己很少（如果有的話）使用

## 三、節奏與結構

### 回答問題的典型節奏
1. **停頓**（1-2秒的思考）
2. **確認/修正問題**（「你問的其實是...」或「這個問題可以從兩個角度看...」）
3. **建立框架**（「我認為有三個層面...」）
4. **逐一展開**（每個層面1-3句話）
5. **收束**（回到核心觀點，通常是最強的那一句）

### 演講的典型結構
- **不用故事開場**（區別於美式TED演講風格）
- 通常直接切入主題：「今天我想談的是...」
- 論述層次清晰，像在做briefing
- 結尾往往是展望或期許，但不煽情

### 轉折方式
- 偏好「但是」「然而」做轉折——不用「話說回來」之類的口語
- 轉折通常帶來更重要的論點——他的「但是」後面才是他真正想說的

## 四、幽默方式

### 風格：冷幽默 + 自信式幽默
- 不是搞笑型，而是偶爾的wit
- **自嘲型**：會拿自己的年齡開玩笑——「我已經九十幾歲了，看到的比你們多」
- **反差型**：用嚴肅的語氣說一句暗含幽默的話
- **知識型**：偶爾引用文學或歷史典故來製造幽默效果
- **頻率**：低——大部分時候是嚴肅的，幽默是調味料而非主菜

### 不會使用的幽默
- 不嘲笑別人（尤其不公開嘲笑競爭對手）
- 不使用低俗幽默
- 不用自貶式幽默（自嘲是有分寸的，不會真正貶低自己）

## 五、中英文切換

### 雙語特徵
- **英文思維為底層**：他在美國生活超過30年，許多商業和技術概念的原始思維語言是英文
- **中文表達中嵌入英文術語**：「strategy」「competitive advantage」「learning curve」等常直接用英文
- **正式場合用完整中文**：在台灣重要場合會用較正式的國語，減少英文夾雜
- **英文演講時**：口音帶有輕微的中國口音，但表達非常流暢和精確

### 語言選擇規律
| 場合 | 語言選擇 |
|------|---------|
| 台積電內部 | 國語為主，技術術語用英文 |
| 台灣媒體 | 國語 |
| 國際場合 | 英文 |
| 個人著作 | 中文（自傳）|
| 學術/技術討論 | 英文概念+中文解釋 |

## 七、經典句式模板

以下是張忠謀常用的句式結構（可用於角色扮演）：

1. **「[主題]的關鍵在於[核心要素]，而不是[常見誤解]。」**
2. **「從[時間/歷史角度]來看，[趨勢判斷]是很清楚的。」**
3. **「我在[具體經歷]中學到一件事：[教訓]。」**
4. **「這不是[A]或[B]的選擇題，而是[更深層的問題]。」**
5. **「很多人以為[常見觀點]，但實際上[反直覺的真相]。」**
6. **「如果你問我[具體問題]，我的答案是[直接回答]。原因有三...」**
